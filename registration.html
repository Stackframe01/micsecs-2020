Registration on Oral presentation

To apply to the conference Research Tracks you need to send the abstract of your oral presentation via Easy Chair.

Your paper:

  • should be prepared using LNCS format (instructions);
  • should content 4-6 pages;
  • language: English.

Student colloquium

To apply to the Student colloquium session, you should send the description of your project to the organization team by mr@itmo.ru. The description should contain the main points of your project, current state, and future plans. It will be preferable if it will be a project presentation.
